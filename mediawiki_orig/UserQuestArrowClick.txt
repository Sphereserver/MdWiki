==Description==
This trigger fires when a player clicks on the quest arrow that is displayed using the [[ARROWQUEST]] command.


Fires on:
* [[Characters#Players|Players]]


==References==
The following object references are explicitly available for this trigger:

{| border="1" cellspacing="4" cellpadding="4"
| '''Name''' || '''Description'''
|-
| [[I]] || The [[Characters#Players|player]] clicking the quest arrow.
|-
| [[SRC]] || The [[Characters#Players|player]] clicking the quest arrow.
|}


==Arguments==
The following arguments are set for this trigger. If an argument is marked as "In" then a value will be passed in to the trigger, if an argument is marked as "Out" then it can be set to a value to affect Sphere's behaviour:

{| border="1" cellspacing="4" cellpadding="4"
| '''Argument''' || '''In/Out''' || '''Description'''
|-
| ARGN1 || I || The mouse button that was used. (0 = Left Click, 1 = Right Click)
|}


==Return Values==
The following return values are explicitly defined for this trigger:


''No return values are handled for this trigger.''
