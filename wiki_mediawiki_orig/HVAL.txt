=DESCRIPTION=
HVAL is the [http://en.wikipedia.org/wiki/Hexadecimal Hexadecimal] version of [http://wiki.sphere.torfo.org/index.php?title=EVAL EVAL], it returns the given values as HEX and it's the default numerical system used by Sphere and the one that will be used to store the numerical values you give in most variable types (TAG,VAR,LOCAL...).

=USAGE=
<spherescript "darkblue">[FUNCTION TestHVAL]
LOCAL.str=<str>
Sysmessage=My real STR is <STR> and when it's stored Sphere uses this value <LOCAL.str>
Sysmessage=I can also get this value using <HVAL <STR>></spherescript>
