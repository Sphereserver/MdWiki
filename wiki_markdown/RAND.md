# RAND

This function returns a random number.

## Ficha

|              |             |
|--------------|-------------|
| **Function** | **RAND**    |
| **Type**     | System      |
| **Access**   | Read        |
| **Sphere X** | Yes         |

## Syntax

- `RAND(x)`: Returns a random value between 0 and `x - 1`.
- `RAND(min, max)`: Returns a random value between `min` and `max`.
