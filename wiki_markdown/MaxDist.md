## Description

This property defines the maximum spawn distance for a spawn item. For items and characters, it's the maximum distance away from the spawn point when created. For NPCs, it's also the maximum distance they can wander from the spawn. It can also be accessed via `MOREZ`.

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **MaxDist**     |
| **Type**     | Spawn Item      |
| **Access**   | Read/Write      |
| **Sphere X** | Yes             |

## Syntax

`[spawn_item].MaxDist = <distance>`
