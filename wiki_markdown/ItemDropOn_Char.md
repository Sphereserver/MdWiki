## Description

This trigger fires when a character drops an item is dropped on to a
character.

Fires on:

-   [Characters](Characters "wikilink")

## References

The following object references are explicitly available for this
trigger:

  ------------------------- -------------------------------------------------------------------------
  **Name**                  **Description**
  [ACT](ACT "wikilink")     The [item](Items "wikilink") being dropped.
  [ARGO](ARGO "wikilink")   The [character](Characters "wikilink") the item is being dropped on to.
  [I](I "wikilink")         The [character](Characters "wikilink") dropping the item.
  ------------------------- -------------------------------------------------------------------------

## Arguments

The following arguments are set for this trigger. If an argument is
marked as \"In\" then a value will be passed in to the trigger, if an
argument is marked as \"Out\" then it can be set to a value to affect
Sphere\'s behaviour:

*No arguments are set for this trigger.*

## Return Values {#return_values}

The following return values are explicitly defined for this trigger:

  ------------------ ---------------------------------------
  **Return Value**   **Description**
  1                  Prevents the item from being dropped.
  ------------------ ---------------------------------------
