# ADD

This command allows adding items to the world.

## Ficha

|              |             |
|--------------|-------------|
| **Command**  | **ADD**     |
| **Type**     | System      |
| **Access**   | Write       |
| **Sphere X** | Yes         |

## Syntax

`ADD <item_defname>[, <amount>]`

## Notes
- Prompts the client to target a location to add the specified item at.
- Accepts a second parameter for the amount of the item to create. Example: `.add i_gold, 500`.
