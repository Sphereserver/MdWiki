## Description

This document defines the `House_Priv` defnames used to control privileges within multi (house) objects.

## Definitions

| Name             | Value | Description                  |
|------------------|-------|------------------------------|
| `HP_NONE`        | 0     | No privileges                |
| `HP_OWNER`       | 1     | Owner privileges             |
| `HP_COOWNER`     | 2     | Co-owner privileges          |
| `HP_FRIEND`      | 3     | Friend privileges            |
| `HP_ACCESSONLY`  | 4     | Access only privileges       |
| `HP_BAN`         | 5     | Banned from house            |
| `HP_VENDOR`      | 6     | Vendor privileges            |
| `HP_GUILD`       | 7     | Guild member privileges      |
