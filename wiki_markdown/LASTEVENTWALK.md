## Description

This read-only client property retrieves the time in ticks (`SERV.TIME`) of the last allowed walk request from the client.

## Ficha

|              |                    |
|--------------|--------------------|
| **Property** | **LASTEVENTWALK**  |
| **Type**     | Client             |
| **Access**   | Read-Only          |
| **Sphere X** | Yes                |

## Syntax

`<CLIENT.LASTEVENTWALK>`

## Return Values

-   Returns the time in ticks (`SERV.TIME`) of the last allowed walk request.
