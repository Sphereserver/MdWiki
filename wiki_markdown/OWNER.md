## Description

This property gets or sets the owner of an NPC. It can be set directly to an NPC's UID, without the need for custom functions.

## Ficha

|              |                               |
|--------------|-------------------------------|
| **Property** | **OWNER**                     |
| **Type**     | NPC                           |
| **Access**   | Read/Write                    |
| **Sphere X** | Yes                           |

## Notes
- Setting `OWNER` to `0` will empty the house value for list-based values (Coowners, Friends, Bans, Vendors, Accesses).
