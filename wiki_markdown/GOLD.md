# GOLD

This property gets or sets the amount of gold a character has.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **GOLD**    |
| **Type**     | Character   |
| **Access**   | Read/Write  |
| **Sphere X** | Yes         |

## Notes
- A limit of 25,000,000 has been added to `GOLD` to prevent server hangs or crashes with excessively high values.
