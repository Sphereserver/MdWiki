## Description

This property sets an object's timer to elapse after a given number of milliseconds.

## Ficha

|              |               |
|--------------|---------------|
| **Property** | **TIMERMS**   |
| **Type**     | Object        |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[object].TIMERMS = <milliseconds>`

## Parameters

-   `<milliseconds>`: The time in milliseconds after which the object's timer will elapse.
