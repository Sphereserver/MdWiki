## Description

This function returns the position of a given character UID in the friend list of a multi (house).

## Ficha

|              |                 |
|--------------|-----------------|
| **Function** | **GetFriendPos** |
| **Type**     | Multi (House)   |
| **Access**   | Read-Only       |
| **Sphere X** | Yes             |

## Syntax

`<[multi].GetFriendPos <char_uid>>`

## Parameters

-   `<char_uid>`: The UID of the character to find in the friend list.

## Return Values

-   Returns the 0-based position of the character in the list.
-   Returns -1 if the character is not found.
