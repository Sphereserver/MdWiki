## Description

This property gets or sets the length of time before the item's timer expires, in tenths of a second. This property is now **DEPRECATED**.

## Ficha

|              |               |
|--------------|---------------|
| **Property** | **TIMERD**    |
| **Type**     | Item/Character |
| **Access**   | Read/Write    |
| **Sphere X** | Yes (Deprecated) |

## Notes
- This property is deprecated. A warning will be shown each time it's used.
- `TIMERT` should be used instead for timer functionality.
