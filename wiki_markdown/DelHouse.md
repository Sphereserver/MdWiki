## Description

This function deletes a house from a player's list of owned houses.

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **DelHouse**  |
| **Type**     | Character     |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[char].DelHouse <uid>`

## Parameters

-   `<uid>`: The UID of the house to delete from the list. Use `-1` to clear the entire list (does not delete the physical house).
