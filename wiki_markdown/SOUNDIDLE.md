# SOUNDIDLE

This CharDef property defines the sound ID for a character's idle action.

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **SOUNDIDLE**   |
| **Type**     | Character (CharDef) |
| **Access**   | Read/Write      |
| **Sphere X** | Yes             |

## Notes
- If set, this property overrides the generic `SOUND` property for idle actions.
- Set to `-1` to prevent this action-related sound from being played.
