# SOUNDNOTICE

This CharDef property defines the sound ID for a character's notice action.

## Ficha

|              |                   |
|--------------|-------------------|
| **Property** | **SOUNDNOTICE**   |
| **Type**     | Character (CharDef) |
| **Access**   | Read/Write        |
| **Sphere X** | Yes               |

## Notes
- If set, this property overrides the generic `SOUND` property for notice actions.
- Set to `-1` to prevent this action-related sound from being played.
