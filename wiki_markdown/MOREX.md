## Description

`MOREX` is a generic property that stores additional data for items, often representing an X-coordinate or other numerical values.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **MOREX**   |
| **Type**     | Item        |
| **Access**   | Read/Write  |
| **Sphere X** | Yes         |

## Notes
- For `t_spellbook` items, `MOREX` was previously used to store the maximum spells, but this is now handled by `TDATA4`.
