## Description

This property turns the character to face a specified object or coordinates.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **FACE**    |
| **Type**     | Character   |
| **Access**   | Write       |
| **Sphere X** | Yes         |

## Syntax
`[object].FACE [object_uid]`
`[object].FACE [p.x],[p.y],[p.z]`

## Notes
- Admits coordinates instead of UID in X branch.
