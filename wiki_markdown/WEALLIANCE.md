# WEALLIANCE

This function sets or checks our guild's alliance status with another guild. It works exactly in the same way as `WEWAR` but used to set alliances instead of wars.

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **WEALLIANCE** |
| **Type**     | Guild         |
| **Access**   | Read/Write    |
| **Sphere X** | Yes           |

## Syntax

`[guild].WEALLIANCE <guild_uid>`

## Notes
- This function is similar to `WEWAR` but specifically for managing alliances.
