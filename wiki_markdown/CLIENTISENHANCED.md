## Description

This command is used to check if the client is an Enhanced Client. It was previously known as `CLIENTISSA`.

## Ficha

|              |             |
|--------------|-------------|
| **Command**  | **CLIENTISENHANCED** |
| **Type**     | Client      |
| **Access**   | Read        |
| **Sphere X** | Yes         |

## Syntax

`<CLIENTISENHANCED>`

## Return Values

- Returns 1 if the client is an Enhanced Client, 0 otherwise.
