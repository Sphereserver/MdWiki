# CLIENTISENHANCED

This command checks if the client is an Enhanced Client. It was previously named `CLIENTISSA`.

## Ficha

|              |                     |
|--------------|---------------------|
| **Command**  | **CLIENTISENHANCED** |
| **Type**     | Client              |
| **Access**   | Read                |
| **Sphere X** | Yes                 |

## Syntax

`[client].CLIENTISENHANCED`

## Notes
- This command was previously named `CLIENTISSA`.
