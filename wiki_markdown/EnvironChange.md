## Description

This trigger fires when the environment (light, season, weather) changes
for a character, or when the character moves in to a new sector.

Fires on:

-   [Characters](Characters "wikilink")

## References

The following object references are explicitly available for this
trigger:

  ----------------------- ----------------------------------------------------------------------------
  **Name**                **Description**
  [SRC](SRC "wikilink")   The [character](Characters "wikilink") affected by the environment change.
  ----------------------- ----------------------------------------------------------------------------

## Arguments

The following arguments are set for this trigger. If an argument is
marked as \"In\" then a value will be passed in to the trigger, if an
argument is marked as \"Out\" then it can be set to a value to affect
Sphere\'s behaviour:

*No arguments set for this trigger.*
