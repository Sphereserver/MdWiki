## Description

This item property was used to get or set the maximum number of charges on an item. It has been removed and superseded by `USESMAX`.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **CHARGESMAX** |
| **Type**     | Item        |
| **Access**   | Read/Write  |
| **Sphere X** | Removed     |

## Notes
- `CHARGESMAX` has been removed and should no longer be used. Please use `USESMAX` instead.
