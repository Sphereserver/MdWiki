# CHARGESMAX

This item property previously stored the maximum number of charges an item could have. It has been removed and replaced by `USESMAX`.

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **CHARGESMAX**  |
| **Type**     | Item            |
| **Access**   | Read/Write      |
| **Sphere X** | No (Removed)    |

## Notes
- This property was only partially supported and has been removed. Use `USESMAX` instead.
