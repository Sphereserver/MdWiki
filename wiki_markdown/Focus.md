## Description

The Focus skill passively increases stamina and mana regeneration. This skill is automatically used when `FEATURE_AOS_UPDATE_B` is enabled.

## Ficha

|              |             |
|--------------|-------------|
| **Skill**    | **Focus**   |
| **Sphere X** | Yes         |

## Behavior

-   **Stamina Gain**: Increases the amount of stamina gained by 1 for each 10% points of skill value.
-   **Mana Gain**: Increases the amount of mana gained by 1 for each 20% points of skill value.
-   **Activation**: Automatically used when the `FEATURE_AOS_UPDATE_B` INI setting is enabled.
