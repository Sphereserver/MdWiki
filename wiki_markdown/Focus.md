# Focus

The Focus skill passively increases stamina and mana regeneration.

## Ficha

|              |             |
|--------------|-------------|
| **Skill**    | **Focus**   |
| **Type**     | Character   |
| **Sphere X** | Yes         |

## Behavior
- The Focus skill is used automatically when `FEATURE_AOS_UPDATE_B` is enabled.
- It passively increases stamina gained by 1 for each 10% points of skill value.
- It passively increases mana gained by 1 for each 20% points of skill value.
