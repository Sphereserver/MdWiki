## Description

This property refers to the character or item that is related to the action the current character is performing.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **ACT**     |
| **Type**     | Character/Item |
| **Access**   | Read/Write  |
| **Sphere X** | Yes         |

## Notes
- `ACT` is now stored in save files to ensure actions are not broken during server restarts.
