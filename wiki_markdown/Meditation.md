## Description

The Meditation skill now considers the `EFFECT` property of the skill definition.

## Ficha

|              |               |
|--------------|---------------|
| **Skill**    | **Meditation** |
| **Sphere X** | Yes           |

## Behavior

-   If the `EFFECT` property is present in the skill definition, its value will be used in calculations.
-   If the `EFFECT` property is not present, a default value of `1` will be used.
