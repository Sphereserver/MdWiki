## Description

This trigger fires when a character is responsible for an item being
created, before the item is placed in the world.

Fires on:

- [Characters](./Characters.md)

## References

The following object references are explicitly available for this
trigger:

|  |  |
|----|----|
| **Name** | **Description** |
| [ACT](./ACT.md) | The [item](./Items.md) used for create the [item](./Items.md). |
|  |  |
| [ARGO](./ARGO.md) | The [item](./Items.md) being created. |
| [I](./I.md) | The [character](./Characters.md) responsible for the item's creation. |

## Arguments

The following arguments are set for this trigger. If an argument is
marked as "In" then a value will be passed in to the trigger, if an
argument is marked as "Out" then it can be set to a value to affect
Sphere's behaviour:

*No arguments are set for this trigger.*
