## Description

This function adds a character UID to the co-owner list of a multi (house).

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **AddCoowner** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[multi].AddCoowner <char_uid>`

## Parameters

-   `<char_uid>`: The UID of the character to add as a co-owner.
