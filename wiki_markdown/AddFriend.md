## Description

This function adds a character UID to the friend list of a multi (house).

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **AddFriend** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[multi].AddFriend <char_uid>`

## Parameters

-   `<char_uid>`: The UID of the character to add to the friend list.
