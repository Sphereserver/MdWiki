## Description

This function removes a vendor UID from the vendor list of a multi (house).

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **DelVendor** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[multi].DelVendor <vendor_uid>`

## Parameters

-   `<vendor_uid>`: The UID of the vendor to remove from the vendor list. Use `-1` to clear the entire list.
