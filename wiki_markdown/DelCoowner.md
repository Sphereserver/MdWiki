## Description

This function deletes a character UID from the co-owner list of a multi (house).

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **DelCoowner** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[multi].DelCoowner <char_uid>`

## Parameters

-   `<char_uid>`: The UID of the character to delete from the co-owner list. Use `-1` to clear the entire list.
