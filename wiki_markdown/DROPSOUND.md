## Description

This item property allows overriding the default sound played when an item is dropped.

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **DROPSOUND** |
| **Type**     | Item            |
| **Access**   | Read/Write      |
| **Sphere X** | Yes             |

## Syntax

`[item].DROPSOUND = <sound_id>`

## Notes
- This property works in conjunction with `PICKUPSOUND` to customize item sound effects.
