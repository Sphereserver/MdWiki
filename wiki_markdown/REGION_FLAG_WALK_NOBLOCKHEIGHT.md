## Description

This region flag ignores a character's height for walkchecks within the region. It functions similarly to `MT_NOBLOCKHEIGHT` but applies at the region level. Use with caution!

## Ficha

|              |                             |
|--------------|-----------------------------|
| **Flag**     | **REGION_FLAG_WALK_NOBLOCKHEIGHT** |
| **Value**    | `040000`                    |
| **Type**     | Region                      |
| **Sphere X** | Yes                         |

## Notes
- This flag can cause characters to walk under ceilings that are too low for their height.
