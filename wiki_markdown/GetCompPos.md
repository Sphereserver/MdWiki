## Description

This function returns the position of a given component UID in the components list of a multi (house).

## Ficha

|              |                 |
|--------------|-----------------|
| **Function** | **GetCompPos**  |
| **Type**     | Multi (House)   |
| **Access**   | Read-Only       |
| **Sphere X** | Yes             |

## Syntax

`<[multi].GetCompPos <item_uid>>`

## Parameters

-   `<item_uid>`: The UID of the component to find in the list.

## Return Values

-   Returns the 0-based position of the component in the list.
-   Returns -1 if the component is not found.
