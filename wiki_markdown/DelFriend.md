## Description

This function deletes a character UID from the friend list of a multi (house).

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **DelFriend** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[multi].DelFriend <char_uid>`

## Parameters

-   `<char_uid>`: The UID of the character to delete from the friend list. Use `-1` to clear the entire list.
