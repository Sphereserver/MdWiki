## Description

This property sets the maximum spawn time in minutes for a spawn item. It can also be accessed via `MOREY`.

## Ficha

|              |               |
|--------------|---------------|
| **Property** | **TimeHi**    |
| **Type**     | Spawn Item    |
| **Access**   | Read/Write    |
| **Sphere X** | Yes           |

## Syntax

`[spawn_item].TimeHi = <minutes>`
