## Description

`MOREZ` is a generic property that stores additional data for items, often representing a Z-coordinate or other numerical values.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **MOREZ**   |
| **Type**     | Item        |
| **Access**   | Read/Write  |
| **Sphere X** | Yes         |

## Notes
- For `t_spellbook` items, `MOREZ` was previously used to store the spell offset, but this is now handled by `TDATA3`.
