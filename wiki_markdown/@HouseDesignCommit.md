# @HouseDesignCommit

This trigger fires when the character commits a new house design.

## Ficha

|              |                      |
|--------------|----------------------|
| **Trigger**  | **@HouseDesignCommit** |
| **Type**     | Character            |
| **Access**   | Read                 |
| **Sphere X** | Yes                  |

## Arguments

The following arguments are set for this trigger:

| Argument   | Description                                            |
|------------|--------------------------------------------------------|
| `local.MAXZ` | (Read Only) Returns the Z value of the top placed item. |

## Return Values

None.
