## Description

Spellbooks are items that can store and allow access to spells. They have unique properties related to how spells are managed within them.

## Ficha

|              |             |
|--------------|-------------|
| **Type**     | t_spellbook |
| **Sphere X** | Yes         |

## Notes
- Spellbooks now store spell offset and maximum spells using `TDATA3` and `TDATA4` instead of `MOREZ` and `MOREX`.
