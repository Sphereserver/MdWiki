## Description

This function checks whether an object has a given `ComponentProps` property.

## Ficha

|              |                        |
|--------------|------------------------|
| **Function** | **HasComponentProps**  |
| **Type**     | Object                 |
| **Access**   | Read-Only              |
| **Sphere X** | Yes                    |

## Syntax

`<[object].HasComponentProps <propid>>`

## Parameters

-   `<propid>`: The ID of the `ComponentProps` to check. Defnames for `ComponentProps` IDs are found in `core/defs_component_props.scp`.
