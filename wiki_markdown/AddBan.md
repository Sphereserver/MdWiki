## Description

This function adds a character UID to the ban list of a multi (house).

## Ficha

|              |           |
|--------------|-----------|
| **Function** | **AddBan** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only |
| **Sphere X** | Yes       |

## Syntax

`[multi].AddBan <char_uid>`

## Parameters

-   `<char_uid>`: The UID of the character to add to the ban list.
