## Description

This function regenerates the comps of a multi (house) based on its `MULTIDEF` definition. It does not delete current comps nor recode/remove current keys.

## Ficha

|              |                    |
|--------------|--------------------|
| **Function** | **GenerateBaseComps** |
| **Type**     | Multi (House)      |
| **Access**   | Write-Only         |
| **Sphere X** | Yes                |

## Syntax

`[multi].GenerateBaseComps`
