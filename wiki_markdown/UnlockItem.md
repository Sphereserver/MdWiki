## Description

This function removes an item UID from the lockdown list of a multi (house).

## Ficha

|              |                 |
|--------------|-----------------|
| **Function** | **UnlockItem**  |
| **Type**     | Multi (House)   |
| **Access**   | Write-Only      |
| **Sphere X** | Yes             |

## Syntax

`[multi].UnlockItem <item_uid>`

## Parameters

-   `<item_uid>`: The UID of the item to remove from the lockdown list. Use `-1` to clear the entire list.
