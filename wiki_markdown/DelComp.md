## Description

This function deletes a component (item) with the given UID from a multi (house).

## Ficha

|              |             |
|--------------|-------------|
| **Function** | **DelComp** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only  |
| **Sphere X** | Yes         |

## Syntax

`[multi].DelComp <item_uid>`

## Parameters

-   `<item_uid>`: The UID of the item to delete from the comps list. Use `-1` to clear the entire list.
