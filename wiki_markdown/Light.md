## Description

This property refers to the light level of a character or a sector.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **LIGHT**   |
| **Type**     | Character/Sector |
| **Access**   | Read/Write  |
| **Sphere X** | Yes         |

## Notes
- If the client's `LIGHT` property was set, the `Night Sight` flag was previously ignored when calculating the light level to send to the client. This has been fixed.
