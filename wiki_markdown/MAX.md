## Description

This intrinsic function returns the greater of two numeric arguments.

## Ficha

|              |            |
|--------------|------------|
| **Function** | **MAX**    |
| **Type**     | Numeric    |
| **Access**   | Read-Only  |
| **Sphere X** | Yes        |

## Syntax

`<MAX <value1>, <value2>>`

## Parameters

-   `<value1>`: The first numeric argument.
-   `<value2>`: The second numeric argument.

## Return Values

-   Returns the greater of `<value1>` and `<value2>`.
