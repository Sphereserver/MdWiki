## Description

This command outputs the tags stored in the Base Def section (Chardef/Itemdef...) of an object.

## Ficha

|              |               |
|--------------|---------------|
| **Command**  | **BASETAGLIST** |
| **Type**     | Object        |
| **Access**   | Read-Only     |
| **Sphere X** | Yes           |

## Syntax

`[object].BASETAGLIST`
