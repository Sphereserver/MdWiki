## Description

This function adds a component (item) with the given UID to a multi (house).

## Ficha

|              |             |
|--------------|-------------|
| **Function** | **AddComp** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only  |
| **Sphere X** | Yes         |

## Syntax

`[multi].AddComp <item_uid>`

## Parameters

-   `<item_uid>`: The UID of the item to add as a comp.
