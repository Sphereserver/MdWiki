# SOUND

This property plays a sound from this object (character, item).

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **SOUND**   |
| **Type**     | Character, Item |
| **Access**   | Read/Write  |
| **Sphere X** | Yes         |

## Syntax

`[object].SOUND <sound_id>, [repeat]`

## Notes
- `SOUND` accepts a sound ID as a parameter. Use `BARK` for sound types.
