## Description

This function adds a vendor UID to the vendor list of a multi (house).

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **AddVendor** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[multi].AddVendor <vendor_uid>`

## Parameters

-   `<vendor_uid>`: The UID of the vendor to add to the vendor list.
