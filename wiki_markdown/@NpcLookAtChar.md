# @NpcLookAtChar

This trigger fires when the NPC looks at a character.

## Ficha

|              |                    |
|--------------|--------------------|
| **Trigger**  | **@NpcLookAtChar** |
| **Type**     | Character (NPC)    |
| **Access**   | Read               |
| **Sphere X** | Yes                |

## Notes
- This trigger may now fire more frequently due to NPCs performing more AI checks ("looking around") while wandering.
