## Description

This command outputs the properties stored in the Base Def section (Chardef/Itemdef...) of an object.

## Ficha

|              |                |
|--------------|----------------|
| **Command**  | **BASEPROPLIST** |
| **Type**     | Object         |
| **Access**   | Read-Only      |
| **Sphere X** | Yes            |

## Syntax

`[object].BASEPROPLIST`
