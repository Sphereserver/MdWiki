## Description

This property represents a virtual gold currency, functioning similarly to `GOLD` when `FEATURE_TOL_VIRTUALGOLD` is enabled.

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **VirtualGold** |
| **Type**     | Character       |
| **Access**   | Read/Write      |
| **Sphere X** | Yes             |

## Notes
- `VirtualGold` will function as `GOLD` and `<GOLD>` when `FEATURE_TOL_VIRTUALGOLD` is enabled.
