## Description

This function retrieves the resource type (RES_ITEMDEF, RES_SPAWN, etc) from a `defname`.

## Ficha

|              |                    |
|--------------|--------------------|
| **Function** | **RESOURCETYPE**   |
| **Type**     | Defname            |
| **Access**   | Read-Only          |
| **Sphere X** | Yes                |

## Syntax

`<RESOURCETYPE <defname>>`

## Parameters

-   `<defname>`: The `defname` of the resource.

## Notes
- The list of resource type numbers can be found in the X scriptpack in `core/defs.scp`.
