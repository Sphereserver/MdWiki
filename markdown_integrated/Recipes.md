## Description

This page lists various recipe properties and skills that can be used in SphereServer.

## Recipe Skills

The following skills are associated with recipes for crafting:

- `RecipeAlchemy`
- `RecipeBlacksmith`
- `RecipeBowcraft`
- `RecipeCarpentry`
- `RecipeCartography`
- `RecipeCooking`
- `RecipeInscription`
- `RecipeTailoring`
- `RecipeTinkering`
- `RecipeMasonry`
- `RecipeGlassblowing`

These can be used in skillmake definitions to control crafting requirements.
