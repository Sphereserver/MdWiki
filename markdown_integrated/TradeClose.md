## Description

This trigger fires when one player is closing a trade with another
player. The trigger will be fired on both characters.

Fires on:

- [Players](./CharactersPlayers.md)

## References

The following object references are explicitly available for this
trigger:

|  |  |
|----|----|
| **Name** | **Description** |
| [SRC](./SRC.md) | The [player](./CharactersPlayers.md) closing the Trade Window. |
| [ARGO](./ARGO.md) | The other [player](./CharactersPlayers.md). |

The following arguments are set for this trigger. If an argument is

argument is marked as "Out" then it can be set to a value to affect
Sphere's behaviour:
