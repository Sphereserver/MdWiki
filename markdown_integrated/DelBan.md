## Description

This function deletes a character UID from the ban list of a multi (house).

## Ficha

|              |           |
|--------------|-----------|
| **Function** | **DelBan** |
| **Type**     | Multi (House) |
| **Access**   | Write-Only |
| **Sphere X** | Yes       |

## Syntax

`[multi].DelBan <char_uid>`

## Parameters

-   `<char_uid>`: The UID of the character to delete from the ban list. Use `-1` to clear the entire list.
