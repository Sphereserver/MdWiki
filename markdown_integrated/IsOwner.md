## Description

This property checks if a specified UID is the owner of a house.

## Ficha

|              |               |
|--------------|---------------|
| **Property** | **IsOwner**   |
| **Type**     | House (Multi) |
| **Access**   | Read-Only     |
| **Sphere X** | Yes           |

## Syntax

`<[house].IsOwner <uid>>`

## Parameters

-   `<uid>`: The UID of the character to check for ownership.

## Return Values

-   Returns 1 if the specified UID is the owner, 0 otherwise.
