# NEWDUPE

This function clones a character or item with the specified UID.

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **NEWDUPE**   |
| **Type**     | World         |
| **Access**   | Write         |
| **Sphere X** | Yes           |

## Syntax

`SERV.NEWDUPE=<object_uid>`

## Notes

- When an object is cloned with `NEWDUPE`, the Garbage Collection report will print the script file name and the line where the object was created.
