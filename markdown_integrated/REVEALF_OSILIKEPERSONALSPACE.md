## Description

This flag, if set, prevents invisible characters from being revealed when another character enters their personal space. Additionally, no message will be sent to the character who entered the personal space.

|--------------|-------------------------------|
| **Flag**     | **REVEALF_OSILIKEPERSONALSPACE** |
| **Type**     | Character (Invisible)          |
| **Sphere X** | Yes                           |
