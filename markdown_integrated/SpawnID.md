## Description

This property defines the ID of the object(s) a spawn item will create. It can be an item/template of items (for `t_spawn_item`) or a character/template of characters (for `t_spawn_char`).

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **SpawnID**     |
| **Type**     | Spawn Item      |
| **Access**   | Read/Write      |
| **Sphere X** | Yes             |

## Syntax

`[spawn_item].SpawnID = <id>`

## Notes
- Can also be accessed via `MORE` or `MORE1`.
- `MORE2` has different behavior depending on the spawn type.
