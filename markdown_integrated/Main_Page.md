## Sphere 3rd Party Tools {#sphere_3rd_party_tools}

-   [Axis
    II](http://forum.spherecommunity.net/sshare.php?srt=4&uid=603) - GM
    Tool for Sphere that will allow you to place objects in-game,
    spawns, traveling and many other useful functions for shard admins
    and GMs.
-   [vSCP](http://forum.spherecommunity.net/sshare.php?srt=4&prj=3) -
    vSCP is the most complete and up-to-date syntax editor for sphere
    scripting. It does contain syntax highlighting, autocomplete,
    folding markers to specify blocks of code that can expand or
    collapse, bookmarks, autoindent, find/replace/gotoline, help guide
    for all the sphere elements added to your code, and more!
-   [vServerLauncher](http://forum.spherecommunity.net/sshare.php?srt=4&prj=2) -
    Install and run the latest build of sphereserver in a few clicks
    with vServerLauncher. It\'s pretty quick and simple!
-   [SphereService](http://forum.spherecommunity.net/sshare.php?srt=4&prj=5) -
    Relaunch SphereSvr.exe everytime it closes/crashes. Automatically
    runs at windows startup and works in a silent mode minimized to
    Tray.
-   [vCrypter](http://forum.spherecommunity.net/sshare.php?srt=4&prj=4) -
    Type the client version and the tool will calculate the correct UO
    login keys for classic or enhanced clients.
