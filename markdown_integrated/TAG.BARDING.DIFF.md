# TAG.BARDING.DIFF

This tag is used to determine the difficulty of Enticement, Peacemaking, and Provocation skills. If this tag is not set, the old behavior is used.

## Ficha

|              |                     |
|--------------|---------------------|
| **Property** | **TAG.BARDING.DIFF** |
| **Type**     | Character           |
| **Access**   | Read/Write          |
| **Sphere X** | Yes                 |
