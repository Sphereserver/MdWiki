Gets or sets the item type.

## Examples

Ingame commands: `<spherescript>`{=html}.XTYPE T_NEW_TYPE // Set
T_NEW_TYPE as type of targetted item. .XSHOW TYPE // Show the TYPE of
targetted item.`</spherescript>`{=html}

Sphere scripting: `<spherescript>`{=html}TYPE T_NEW_TYPE // Set
T_NEW_TYPE as type of the item. `<TYPE>`{=html} // Return the type of
the item.`</spherescript>`{=html}

## Notes on Specific Types

-   **`t_multi_custom`**: This type can be dynamically assigned to an item, allowing for flexible configuration of custom multi-part items.
