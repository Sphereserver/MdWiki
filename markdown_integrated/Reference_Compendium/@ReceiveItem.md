## Description

This trigger fires when an NPC is given an item from another character, before they have decided whether or not to accept it.

Fires on:

- [NPCs](Characters#NPCs "wikilink")

## References

The following object references are explicitly available for this trigger:

|  |  |
|----|----|
| **Name** | **Description** |
| [ARGO](ARGO "wikilink") | The [item](Items "wikilink") the NPC is being given. |
| [I](I "wikilink") | The [NPC](Characters#NPCs "wikilink") receiving the item. |
| [SRC](SRC "wikilink") | The [character](Characters "wikilink") giving the item away. |

## Arguments

The following arguments are set for this trigger. If an argument is marked as "In" then a value will be passed in to the trigger, if an argument is marked as "Out" then it can be set to a value to affect Sphere's behaviour:

*No arguments are set for this trigger.*

## Return Values

The following return values are explicitly defined for this trigger:

|                  |                          |
|------------------|--------------------------|
| **Return Value** | **Description**          |
| 1                | Returns the item to SRC. |
