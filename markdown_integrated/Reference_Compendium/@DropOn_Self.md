## Description

This trigger fires when an item has an item dropped on to it. For example, when an item is placed inside a backpack this trigger will fire on the backpack.

Fires on:

- [Items](Items "wikilink")

## References

The following object references are explicitly available for this trigger:

|  |  |
|----|----|
| **Name** | **Description** |
| [ARGO](ARGO "wikilink") | The [item](Items "wikilink") being dropped on to the item. |
| [I](I "wikilink") | The [item](Items "wikilink") that is being dropped on to. |
| [SRC](SRC "wikilink") | The [character](Characters "wikilink") dropping the item. |

## Arguments

The following arguments are set for this trigger. If an argument is marked as "In" then a value will be passed in to the trigger, if an argument is marked as "Out" then it can be set to a value to affect Sphere's behaviour:

*No arguments are set for this trigger.*

## Return Values

The following return values are explicitly defined for this trigger:

|  |  |
|----|----|
| **Return Value** | **Description** |
| 1 | Prevents the item from being dropped on to the item. If no CONT is set, item will drop on ground |
