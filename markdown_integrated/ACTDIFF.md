# ACTDIFF

This property holds the difficulty of the character's current action.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **ACTDIFF** |
| **Type**     | Character   |
| **Access**   | Read/Write  |
| **Sphere X** | Yes         |

## Notes
- `ACTDIFF` is now set and cleared in the same situations as `ACTARG1/2/3`, which is at the end of a skill (success, abort, or failure).
