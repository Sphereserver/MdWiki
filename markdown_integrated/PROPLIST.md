## Description

This command outputs a list of all properties currently set on an object. It now also shows properties stored in `ComponentProps`, preceded by the name of the component inside square brackets.

## Ficha

|              |             |
|--------------|-------------|
| **Command**  | **PROPLIST** |
| **Type**     | Object      |
| **Access**   | Read-Only   |
| **Sphere X** | Yes         |

## Syntax

`[object].PROPLIST`
