## Description

This function adds an item UID to the lockdown list of a multi (house).

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **LockItem**  |
| **Type**     | Multi (House) |
| **Access**   | Write-Only    |
| **Sphere X** | Yes           |

## Syntax

`[multi].LockItem <item_uid>`

## Parameters

-   `<item_uid>`: The UID of the item to add to the lockdown list.
