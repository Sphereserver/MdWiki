# DUPE

This function clones the item.

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **DUPE**      |
| **Type**     | Item          |
| **Access**   | Write         |
| **Sphere X** | Yes           |

## Syntax

`[item].DUPE`

## Notes

- When an item is cloned with `DUPE`, the Garbage Collection report will print the script file name and the line where the object was created.
