## Combat Settings

Sphere provides several settings in Sphere.ini to control combat behavior.

### CombatDamageEra

The `CombatDamageEra` setting allows you to switch between different damage formulas.

```
If `CombatDamageEra` is set to 0, the old HitChance formula is used.

```
### CombatHitChanceEra
```

The `CombatHitChanceEra` setting allows you to switch between different Hit Chance formulas for physical combat.
```
