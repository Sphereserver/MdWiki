## Description

This item property allows overriding the default sound played when an item is picked up.

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **PICKUPSOUND** |
| **Type**     | Item            |
| **Access**   | Read/Write      |
| **Sphere X** | Yes             |

## Syntax

`[item].PICKUPSOUND = <sound_id>`

## Notes
- This property works in conjunction with `DROPSOUND` to customize item sound effects.
