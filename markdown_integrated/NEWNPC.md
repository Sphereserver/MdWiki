# NEWNPC

This function creates a new character.

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **NEWNPC**    |
| **Type**     | World         |
| **Access**   | Write         |
| **Sphere X** | Yes           |

## Syntax

`SERV.NEWNPC=<character_defname>`

## Notes

- When a character is created with `NEWNPC`, the Garbage Collection report will print the script file name and the line where the object was created.
