## Description

This trigger fires when a resource is being gathered. (Equivalent
trigger [@ResourceFound](@ResourceFound "wikilink") on
[Regions](Regions "wikilink"))

Fires on:

- [Characters](Characters "wikilink")

## References

The following object references are explicitly available for this
trigger:

|  |  |
|----|----|
| **Name** | **Description** |
| [ARGO](ARGO "wikilink") | The [worldgem bit](Items "wikilink") that represents the resource in the world. |
| [I](I "wikilink") | The [region resource](REGIONRESOURCE "wikilink") being gathered. |
| [SRC](SRC "wikilink") | The [character](Characters "wikilink") gathering the resource. |

## Arguments

The following arguments are set for this trigger. If an argument is
marked as "In" then a value will be passed in to the trigger, if an
argument is marked as "Out" then it can be set to a value to affect
Sphere's behaviour:

*No arguments are set for this trigger.*

## Return Values

The following return values are explicitly defined for this trigger:

|  |  |
|----|----|
| **Return Value** | **Description** |
| 1 | Sets the amount of resource to zero, preventing it from being gathered. |
