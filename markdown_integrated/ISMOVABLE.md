## Description

This property allows to verify if the item is movable or not.

## Ficha

|              |             |
|--------------|-------------|
| **Property** | **ISMOVABLE** |
| **Type**     | Item        |
| **Access**   | Read        |
| **Sphere X** | Yes         |

## Syntax

`<I.ISMOVABLE>`

## Example

`IF (<I.ISMOVABLE> == 1) SYSMESSAGE "This item is movable."`
