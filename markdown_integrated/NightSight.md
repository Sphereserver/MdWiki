## Description

This flag, when active, allows a character to see better in low-light conditions.

## Ficha

|              |             |
|--------------|-------------|
| **Flag**     | **Night Sight** |
| **Type**     | Character   |
| **Sphere X** | Yes         |

## Notes
- The `Night Sight` flag was previously ignored when the client's `LIGHT` property was set. This has been fixed to ensure proper light level calculation.
