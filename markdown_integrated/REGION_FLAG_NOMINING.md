## Description

This region flag prevents mining within a specific region.

## Ficha

|              |                      |
|--------------|----------------------|
| **Flag**     | **REGION_FLAG_NOMINING** |
| **Value**    | `020000`             |
| **Type**     | Region               |
| **Sphere X** | Yes                  |

## Notes
- To prevent mining in a region, add this flag to the region's `FLAGS` property.
