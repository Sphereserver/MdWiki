# PLEVEL

This property gets or sets the account's privilege level.

## Ficha

|              |                 |
|--------------|-----------------|
| **Property** | **PLEVEL**      |
| **Type**     | Account         |
| **Access**   | Read/Write      |
| **Sphere X** | Yes             |

## Notes
- Clients with an account `PLEVEL` of 2 or greater (starting from Counselor priv) now log out instantly. Previously, this only occurred if GM mode was activated, regardless of the `PLEVEL`.
