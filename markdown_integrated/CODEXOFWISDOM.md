## Description

This function opens the client's Codex of Wisdom menu at a given topic ID.

## Ficha

|              |                   |
|--------------|-------------------|
| **Function** | **CODEXOFWISDOM** |
| **Type**     | Client            |
| **Access**   | Write-Only        |
| **Sphere X** | Yes               |

## Syntax

`[char].CODEXOFWISDOM <TopicID>[, <ForceOpen>]`

## Parameters

-   `<TopicID>`: The ID of the topic to open (range: 1~176).
-   `<ForceOpen>`: (Optional) `0` or `1`. If `1`, forces the menu open.
