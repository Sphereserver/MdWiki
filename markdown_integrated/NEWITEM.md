# NEWITEM

This function creates a new item.

## Ficha

|              |               |
|--------------|---------------|
| **Function** | **NEWITEM**   |
| **Type**     | World         |
| **Access**   | Write         |
| **Sphere X** | Yes           |

## Syntax

`SERV.NEWITEM=<item_defname>,<amount>,<container>,<equip>`

## Notes

- When an item is created with `NEWITEM`, the Garbage Collection report will print the script file name and the line where the object was created.
