## Description

This trigger periodically fires in regions that clients are active in.
Unlike the [@CliPeriodic](./CliPeriodic.md) trigger, this trigger
will only fire once.

Fires on:

- [Regions](./Regions.md)

## References

The following object references are explicitly available for this
trigger:

|  |  |
|----|----|
| **Name** | **Description** |
| [I](./I.md) | The [region](./Regions.md) with active client(s). |
| [SRC](./SRC.md) | A [client](./CharactersClients.md) that is inside the region. |

## Arguments

The following arguments are set for this trigger. If an argument is
marked as "In" then a value will be passed in to the trigger, if an
argument is marked as "Out" then it can be set to a value to affect
Sphere's behaviour:

*No arguments are set for this trigger.*
